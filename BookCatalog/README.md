# Каталог за книги

## Описание
"Каталог за книги" е ASP.NET Core MVC приложение, което предоставя пълна функционалност за управление на книги, автори, жанрове и тагове. Потребителите могат да добавят, редактират, преглеждат и изтриват записи.

## Функционалности
- **CRUD операции** за книги, автори, жанрове и тагове
- **Свързване на книги с автори и жанрове**
- **Добавяне на тагове към книги**
- **Качване и показване на изображения за книги, автори и жанрове**

## Технологии
- **ASP.NET Core MVC** – за уеб приложението
- **Entity Framework Core** – за управление на базата данни
- **Microsoft SQL Server** – за съхранение на данни

## Структура на базата данни
Приложението използва следните основни модели:

### Book (Книга)
- `BookId` – Уникален идентификатор
- `Title` – Заглавие на книгата
- `BookInformation` – Информация за книгата
- `AuthorId` – Връзка към автора
- `GenreId` – Връзка към жанра
- `ImagePath` – Път до изображение
- `BookTags` – Колекция от тагове

### Author (Автор)
- `AuthorId` – Уникален идентификатор
- `Name` – Име на автора
- `Bio` – Биография на автора
- `ImagePath` – Път до изображение
- `Books` – Списък с книги на автора

### Genre (Жанр)
- `GenreId` – Уникален идентификатор
- `Name` – Име на жанра
- `Info` – Описание на жанра
- `ImagePath` – Път до изображение
- `Books` – Списък с книги в жанра

### Tag (Таг)
- `TagId` – Уникален идентификатор
- `Name` – Име на тага
- `BookTags` – Списък с книги, асоциирани с този таг

### BookTag (Тагове към книги)
- `BookId` – Връзка към книга
- `TagId` – Връзка към таг

## Инсталация и стартиране
1. **Клониране на хранилището:**
   ```bash
   git clone https://github.com/your-repository-url.git
   cd your-repository-folder
   ```
2. **Конфигуриране на базата данни:**
   - Промени `appsettings.json`, за да добавиш връзка към базата данни.
   ```json
   "ConnectionStrings": {
     "DefaultConnection": "Server=.;Database=BookCatalogDb;Trusted_Connection=True;"
   }
   ```
3. **Изпълнение на миграции:**
   ```bash
   dotnet ef database update
   ```
4. **Стартиране на приложението:**
   ```bash
   dotnet run
   ```
5. **Отваряне в браузър:**
   ```
   http://localhost:5000
   ```
